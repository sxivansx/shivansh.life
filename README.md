# shivansh.life

personal portfolio. minimal, black and white.

## stack

- next.js 16
- tailwind css 4
- framer motion
- geist font

## run locally

```bash
bun install
bun dev
```

## features

- dark mode by default, manual light/dark toggle
- scroll-triggered fade-in animations
- embedded spotify playlist
- github contribution graph
- film grain overlay
- console easter egg
- one piece footer quote
